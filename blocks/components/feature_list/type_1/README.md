# Feature List Component (type_1)

Bulleted feature list with icons.

## Data Structure
```json
{
  "title": "Features",
  "features": [
    { "icon": "checkmark-outline", "text": "Feature description" }
  ]
}
```

## Usage
```json
{
  "type": "component",
  "component": "feature_list/type_1",
  "id": "features",
  "variant": "main",
  "data": { "main": { ... } }
}
```
