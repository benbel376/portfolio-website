# Tech Stack Component (type_1)

Technology stack display with icons.

## Data Structure
```json
{
  "technologies": [
    { "name": "Python", "icon": "logo-python", "category": "Language" }
  ]
}
```

## Usage
```json
{
  "type": "component",
  "component": "tech_stack/type_1",
  "id": "tech-stack",
  "variant": "main",
  "data": { "main": { ... } }
}
```
