# Stats Grid Component (type_1)

Statistics display grid with icons.

## Data Structure
```json
{
  "stats": [
    { "icon": "code-outline", "value": "50+", "label": "Projects" }
  ]
}
```

## Usage
```json
{
  "type": "component",
  "component": "stats_grid/type_1",
  "id": "stats",
  "variant": "main",
  "data": { "main": { ... } }
}
```
