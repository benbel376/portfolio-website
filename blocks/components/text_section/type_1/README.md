# Text Section Component (type_1)

Rich text content section.

## Data Structure
```json
{
  "title": "Section Title",
  "content": "<p>HTML content...</p>",
  "alignment": "left"
}
```

## Usage
```json
{
  "type": "component",
  "component": "text_section/type_1",
  "id": "about-text",
  "variant": "main",
  "data": { "main": { ... } }
}
```
