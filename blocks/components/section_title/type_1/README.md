# Section Title Component (type_1)

Section header with icon and subtitle.

## Data Structure
```json
{
  "icon": "briefcase-outline",
  "title": "Section Title",
  "subtitle": "Section description"
}
```

## Usage
```json
{
  "type": "component",
  "component": "section_title/type_1",
  "id": "section-header",
  "variant": "main",
  "data": { "main": { ... } }
}
```
