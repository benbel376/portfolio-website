# Project Hero Component (type_1)

Project page hero with banner and title.

## Data Structure
```json
{
  "title": "Project Title",
  "subtitle": "Project tagline",
  "banner": "path/to/banner.png",
  "tags": ["Tag1", "Tag2"]
}
```

## Usage
```json
{
  "type": "component",
  "component": "project_hero/type_1",
  "id": "project-hero",
  "variant": "main",
  "data": { "main": { ... } }
}
```
